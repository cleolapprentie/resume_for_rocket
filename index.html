簡單的自我介紹

...

　  高雄人，中央大學法語系畢業，熱愛日本、語言與設計。目前正積極學習設計相關技能與前端語言，近期目標成為能獨當一面的前端工程師。平時也會接觸一些UI/UX的書籍，培養設計思維。在學習Coding的過程中，對程式設計漸漸培養出興趣，所以我立志要成為兼具設計思維與開發能力的前端工程師！

　  具有初階的前端設計基本知識，已掌握基本的HTML/CSS，也懂得如何應用jQuery來添加動畫效果，以及如何打造出響應式網頁。目前正投入學習Sass/SCSS以及如何運用Bootstrap框架。 　　 　　
　　

為什麼想嘗試程式開發？

...

　  在我國高中時期，因緣際會下接觸了 HTML 與 CSS 語法，讓我對前端設計產生了好奇，並許下未來要成為前端設計師之願。然而，高中升大學之後因為課業繁忙，也因為不是相關科系一直沒機會好好正視這塊。當然這些都是藉口，只因為我還沒有下定決心罷了。大學畢業進入一間日商公司上班，做著的並不是自己真正想要工作。日復一日，過著平庸沒有任何漣漪的生活，我不禁問了問自己內心，是否還記得你的初衷？

　  是的，前端設計就是我的初衷。想當初那奮不顧身一切只專注於鑽研語法的我，竟因為輸給了現實的環境而放棄了夢想。為了抓住這個夢想，便開啟了我的自學之路，每天抓緊下班時間，學習前端語言技能。正因為我不是相關科系出身，所以我有覺悟，要加倍努力來彌補我這些年來的空白。而在學習 coding 的過程中，也因此培養出對程式設計的興趣。所以我有了新的目標：成為兼具設計思維與開發能力的前端工程師！

　  我對於喜愛的事物有強烈的熱情與執著，前端工程師是我無論如何也想實現的目標。一旦下定了目標，就會堅持到底，這點我已經做好了覺悟。雖然我在這個領域還很菜，有很多需要學習的地方，但我已踏出了第一步，接下來就是付諸實行。等著我的未來或許辛苦，但我不怕且樂意挑戰，相信對於學習的熱忱與積極性我不會輸給任何人！ 　　 　　
　　

是否已經嘗試在寫程式，目前有碰到什麼瓶頸？

...

　  真正開始著手學習寫網頁的時候是在今年的3月，從 HTML / CSS 基礎開始，目前已經有打造響應式網頁的能力，以及利用 jQuery plugin 讓網頁內容更豐富。而目前正著手學習 Sass/SCSS 與 Bootstrap 框架，提升刻畫網頁的能力。等這些告一段落之後，再轉向 JavaScript 進攻。這是我目前的學習計畫。

　  我都是靠線上資源來學習，從Youtube上的免費資源到Udemy等付費資源。實作時若遇到不懂的地方，再輔以 Google 與大家的前端學習筆記。然而我也意識到一個難題，那就是利用每天下班時間其實不太夠，再加上因為不是本科系，手邊除了線上之外沒有充足的資源，讓我常常覺得自己孤立無援。學習過程中也有累的一天，卻沒有人可以替我加油打氣，因為到目前為止我靠的都是自己。

　  這次偶然在六角學院的線上問答得知火箭隊計畫，我認為這就是我想要的！一個能專心研究前端語言的地方，也是一個能和志同道合的人建立連結的機會。我已經有了破釜沉舟的決心，前陣子毅然決然地跟現在的公司辭職，為的就是要讓自己能全職投入學習。我希望能藉由火箭隊提供的資源，締結同伴，加速達成我的目標。為了能進入火箭隊計劃，我想我已經做好了萬全的準備！
